UC00604 – Programar para a Web (Frontend)

Código: UC00604

Descrição

Esta unidade de competência visa capacitar o formando para definir parâmetros da estrutura de páginas web usando HTML e criar páginas estruturadas utilizando folhas de estilo CSS, aplicando técnicas de programação no lado do cliente. :contentReference[oaicite:1]

Objetivos

  • Compreender estrutura e organização de páginas web.
  • Aplicar HTML para estruturar conteúdos.
  • Usar CSS para estilos e layout.
  • Introdução a script client-side.

Competências Desenvolvidas

  • Estruturar páginas HTML corretamente.
  • Aplicar estilos CSS consistentes e acessíveis.
  • Capacitar programação front-end básica (cliente-side).
